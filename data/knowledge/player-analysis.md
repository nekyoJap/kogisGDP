# 選手分析の方法

## 脚質（きゃくしつ）

選手の戦い方のスタイルを表す。予想の最重要ファクターの一つ。

### 3つの脚質

| 脚質 | 表記 | 説明 | 特徴 |
|-----|------|------|------|
| 逃げ | 逃 | 先行選手。自らレースを引っ張る | 残り1周半〜2周から先頭に立ち、そのまま押し切る。スタミナと脚力が必要 |
| 追い込み | 追 | 番手・三番手で戦う選手 | ライン先頭の選手の後ろにつき、最終直線で差し切る。判断力と瞬発力が重要 |
| 両 | 両 | 自在選手。先行も追い込みもできる | 展開に応じて戦い方を変えられる。器用さが武器 |

### 脚質の見極めポイント

- **B（バック回数）が多い** → 先行力がある
- **競走得点が高く、Bが少ない** → 追い込みで結果を出している
- **決まり手に「逃げ」「捲り」が多い** → 自力型
- **決まり手に「差し」「マーク」が多い** → 追い込み型

## 競走得点の見方

競走得点は選手の実力を測る最も客観的な指標。直近4ヶ月の成績から算出される。

### 得点の目安

| 得点帯 | 級班目安 | レベル |
|--------|---------|--------|
| 120点以上 | SS級 | トップ中のトップ |
| 110〜119点 | S1上位 | 一線級 |
| 100〜109点 | S1〜S2 | S級標準 |
| 90〜99点 | S2〜A1 | 中堅 |
| 80〜89点 | A1〜A2 | A級上位 |
| 70〜79点 | A2〜A3 | A級標準 |

### 得点の使い方

- **同一ライン内の比較**: ライン先頭の得点が高いほど、そのラインは有力
- **級班をまたぐ比較には注意**: S級とA級では対戦相手のレベルが違うため、単純比較できない
- **得点の推移**: 上昇中の選手は好調、下降中は不調のサイン

## 決まり手の理解

競輪の決まり手（勝ち方）を理解することは、レース展開を予想するうえで重要。

### 主な決まり手

| 決まり手 | 説明 | 選手タイプ |
|---------|------|-----------|
| 逃げ | 先頭で逃げ切って1着 | 先行選手 |
| 捲り（まくり） | 後方から一気にまくって1着 | 自力選手 |
| 差し | 最終直線で前の選手を差して1着 | 追い込み選手 |
| マーク | ライン先頭の選手について行き1着 | 追い込み選手 |

### 決まり手と車券の関係

- **逃げが決まりやすいバンク**: 先行選手のラインを中心に買う
- **差しが決まりやすいバンク**: 番手選手（ライン2番手）に注目
- **捲りが決まりやすい展開**: 後方待機の自力選手に注意

## 選手の調子・状態の見方

### 直近成績のチェックポイント

1. **直近4走の着順**: 安定して上位に来ているか
2. **決まり手の傾向**: いつもの戦い方ができているか
3. **競走得点の推移**: 上昇・下降・横ばい

### コンディション要素

- **落車・怪我の有無**: 直近に落車している選手は状態が万全でない可能性
- **連闘（れんとう）**: 短期間に連続出走している場合、疲労に注意
- **ホームバンク**: 自場（ホームバンク）では走り慣れているため有利な傾向

### 選手コメントの読み方

出走前のコメントからも情報が得られる。

- 「脚の状態がいい」「感触がいい」 → 好調アピール
- 「前で勝負」「先行する」 → 先行宣言（ラインの並びに影響）
- 「流れで」「臨機応変に」 → 展開次第で動く意思表示
- 「連れ込む」「番手から」 → 追い込みに徹する宣言

## 選手の特性評価

### 先行意欲

先行選手がどれだけ積極的にレースを引っ張るかの指標。

| 評価 | 意味 |
|-----|------|
| ◎ | 非常に高い。自ら先頭に立ちたがる |
| ○ | 高い。基本的に先行する |
| △ | 場合による。展開次第で先行 |
| × | 低い。先行はあまりしない |

### スタミナ

- 長い距離を先行できる持久力
- スタミナがある先行選手は、早めの仕掛けでも粘れる
- スタミナが低い先行選手は、仕掛けのタイミングが遅くなりがち

### スピード（トップスピード）

- 最高速度の高さ
- 追い込み選手のスピードは、最終直線での差し脚に直結
- 先行選手のスピードは、逃げ切り力に影響

### 競輪IQ

- レース中の判断力、位置取りの巧さ
- IQが高い選手は、不利な展開でも好走する
- 番手選手の競輪IQは特に重要（差すタイミング、コース取り）
