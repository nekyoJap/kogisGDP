# ラインの理解

## ラインとは

ラインとは、レース中に選手同士が組むチーム（隊列）のこと。競輪最大の特徴であり、予想のカギとなる。

### なぜラインを組むのか

競輪では、空気抵抗が非常に大きい。時速60〜70km/hで走る中、先頭を走る選手は後方の選手に比べて大きな体力を消耗する。そのため、選手は隊列を組んで風よけを分担し、チームとして有利なポジションを狙う。

### ラインの基本構成

| 位置 | 役割 | 求められる能力 |
|-----|------|---------------|
| 先頭（自力） | ラインを引っ張る。先行または捲りで勝負 | 脚力、スタミナ、勇気 |
| 番手（2番手） | 先頭の後ろにつく。先頭を守りながら最終直線で差す | 判断力、スピード、ブロック力 |
| 三番手 | ラインの最後尾。番手の後ろから差しを狙う | 瞬発力、コース取り |

## ライン構成の読み方

### ラインを決める要素

#### 1. 地区（地域のつながり）

最も基本的なライン構成要素。同じ地区の選手同士でラインを組む。

| 地区 | 都道府県 |
|-----|---------|
| 北日本 | 北海道、青森、秋田、岩手、山形、宮城、福島 |
| 関東 | 茨城、栃木、群馬、埼玉、東京、千葉、神奈川、山梨、長野、新潟 |
| 南関東 | 千葉、神奈川、静岡 |
| 中部 | 富山、石川、福井、岐阜、愛知、三重 |
| 近畿 | 滋賀、京都、大阪、兵庫、奈良、和歌山 |
| 中国 | 鳥取、島根、岡山、広島、山口 |
| 四国 | 徳島、香川、愛媛、高知 |
| 九州 | 福岡、佐賀、長崎、熊本、大分、宮崎、鹿児島 |

#### 2. 同期（期別のつながり）

競輪学校（日本競輪選手養成所）の同期生は絆が深く、地区を超えてラインを組むことがある。

#### 3. 師弟関係

師匠と弟子の関係もラインに影響する。

#### 4. 過去のレースでの関係

以前に同じラインで走った経験がある選手同士は組みやすい。

### ライン構成の確認方法

- **出走表の府県欄**: 同じ地区の選手をチェック
- **選手コメント**: 「◯◯さんの後ろ」「◯番の番手」など直接的な情報
- **並び予想**: 専門紙やWebサイトに掲載される事前予想

## ラインの強弱判断

### 強いラインの条件

1. **先頭選手の脚力が高い**: 競走得点が高く、先行力がある
2. **番手選手の実力が高い**: 差し脚があり、ブロック力がある
3. **3人以上の長いライン**: 人数が多いほどレースを支配しやすい
4. **連携が取れている**: 同地区・同期など絆が深い

### ラインの並び方パターン

#### 3分戦（3ライン）

最も多いパターン。9車立てで3つのラインに分かれる。

```
例: 1-4-7 / 2-5-8 / 3-6-9
     先行   捲り   追い込み
```

#### 4分戦（4ライン）

ラインが細かく分かれるため、展開が読みにくい。

#### 2分戦（2ライン）

大きく2つに分かれる。力と力のぶつかり合いになりやすい。

### ラインの有利・不利

- **前団（先行ライン）**: レースを支配できるが、後方からの捲りに注意
- **中団**: 前団と後団の動きを見て対応できるが、中途半端になるリスク
- **後団（捲りライン）**: 前の展開を見てから仕掛けられるが、前が固まると届かない

## 特別競走（GI・GP）でのライン

グレードの高いレースでは、普段と異なるライン構成になることがある。

- **オールスター**: 選考方法が特殊で、地区の偏りが生じやすい
- **グランプリ**: 9名しか出走せず、1人で戦う「単騎」が多い
- **東西対抗戦**: 東日本vs西日本で明確に分かれる

## ライン予想の実践手順

1. **出走表で府県を確認**: 同地区の選手をグルーピング
2. **脚質を確認**: 各グループの先行・追い込みのバランスを見る
3. **選手コメントを参照**: 並びの手がかりを得る
4. **ラインの強弱を評価**: 先頭選手の力、ラインの長さ、連携度
5. **有力ラインを2〜3本に絞る**: そこから展開予想へ
