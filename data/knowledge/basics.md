# 競輪の基礎知識

## 競輪とは

競輪は、バンク（すり鉢状の走路）を自転車で走り、着順を競う公営競技である。1レースは通常7〜9名の選手で行われ、周回数はバンクの周長によって異なる。選手同士が「ライン」と呼ばれるチームを組んで戦う点が、他の公営競技にはない最大の特徴である。

## レースの基本ルール

### 周回数とバンク周長

| バンク周長 | 周回数 | 主な競輪場 |
|-----------|--------|-----------|
| 333m | 6周（2,000m） | 前橋、小倉 |
| 335m | 6周（2,010m） | 松戸 |
| 400m | 5周（2,000m） | 大宮、平塚、京王閣 他多数 |
| 500m | 4周（2,000m） | 宇都宮 |

### レース進行

1. **発走**: 号砲とともにスタート。誘導員の後ろにつく
2. **誘導**: 誘導員が先頭を走り、ペースを作る
3. **打鐘（ジャン）**: 残り1周半で鐘が鳴る。誘導員が退避し、ここから本格的な戦いが始まる
4. **最終周回**: 赤板が表示される。ゴールまでの駆け引きが最も激しくなる
5. **ゴール**: フィニッシュラインを通過した順で着順が決定

### 失格・注意

- **斜行**: 急な進路変更で他選手を妨害した場合
- **押圧**: 他選手を押して進路を奪った場合
- **内圏線踏み切り**: バンクの内側のラインを踏み切った場合
- **先頭員追い抜き**: 誘導員退避前に追い抜いた場合

## 車番と枠番

### 車番（しゃばん）

選手に割り当てられる1〜9の番号。ユニフォームとヘルメットの色で識別する。

| 車番 | 色 | 車番 | 色 |
|-----|-----|-----|-----|
| 1 | 白 | 6 | 緑 |
| 2 | 黒 | 7 | 橙（だいだい） |
| 3 | 赤 | 8 | 桃（ピンク） |
| 4 | 青 | 9 | 紫 |
| 5 | 黄 | | |

### 枠番（わくばん）

9車立ての場合、枠番は以下のように振り分けられる。

| 枠番 | 車番 |
|-----|------|
| 1枠 | 1番 |
| 2枠 | 2番 |
| 3枠 | 3番 |
| 4枠 | 4・5番 |
| 5枠 | 6・7番 |
| 6枠 | 8・9番 |

## 級班制度

競輪選手は成績に基づいて6つの級班に分類される。上位ほどレベルが高い。

| 級班 | 説明 | 人数目安 |
|-----|------|---------|
| SS | S級S班。年間最上位の9名が選出される最高峰 | 9名 |
| S1 | S級1班。トップクラスの実力者 | 約200名 |
| S2 | S級2班。S級の中堅層 | 約200名 |
| A1 | A級1班。A級のトップ | 約500名 |
| A2 | A級2班。A級の中堅 | 約500名 |
| A3 | A級3班。新人やA級下位 | 約500名 |

### 級班の入れ替え

- 半年ごと（1月・7月）に成績に基づいて見直し
- S級S班は毎年12月のグランプリ出場者＋賞金上位で決定
- A級からS級への昇班は、A1上位者が対象

## グレード制度

レースにはグレード（格付け）があり、上位ほど出場資格が厳しく、賞金も高い。

| グレード | 名称 | 特徴 |
|---------|------|------|
| GP | KEIRINグランプリ | 年末に行われる最高峰の一戦。S級S班9名で争う |
| GI | G1 | 高松宮記念杯、日本選手権、オールスター等 |
| GII | G2 | サマーナイトフェスティバル、ウィナーズカップ等 |
| GIII | G3 | 各競輪場の記念競輪 |
| FI | F1 | 通常のS級戦 |
| FII | F2 | A級選手のレース |

### 主要GIレース

- **日本選手権競輪（ダービー）**: 5月開催。最も格式が高い
- **高松宮記念杯**: 6月開催。東西対抗戦
- **オールスター競輪**: 8月開催。ファン投票で出場選手決定
- **寬仁親王牌**: 10月開催
- **競輪祭**: 11月開催。グランプリ出場者の最終選考を兼ねる
- **KEIRINグランプリ**: 12月30日開催。年間王者決定戦

## 出走表の読み方

出走表には以下の情報が掲載される。

### 基本情報
- **車番**: 1〜9の番号
- **選手名**: フルネーム
- **府県**: 所属する都道府県（ライン構成の手がかり）
- **期別**: 競輪学校の卒業期（同期の絆がラインに影響）
- **級班**: SS〜A3
- **脚質**: 逃・追・両（レース展開予想の基本）

### 成績データ
- **競走得点**: 直近の成績を数値化したもの。高いほど好成績
- **勝率**: 1着の割合
- **連対率（2連対率）**: 2着以内の割合
- **3連対率**: 3着以内の割合

### 直近成績
- **B（バック回数）**: 残り1周のバックストレッチを先頭で通過した回数。先行力の指標
- **H（ホーム回数）**: 残り半周のホームストレッチを先頭で通過した回数
- **S（スタート回数）**: 初手で先行した回数
