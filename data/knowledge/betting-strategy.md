# 車券戦略

## 車券の種類

競輪には7種類の車券（投票方式）がある。

| 車券 | 正式名称 | 内容 | 的中確率（9車） | 備考 |
|-----|---------|------|---------------|------|
| 3連単 | 三連勝単式 | 1-2-3着を着順通りに当てる | 1/504 | 最も高配当 |
| 3連複 | 三連勝複式 | 1-2-3着を順不同で当てる | 1/84 | 3連単の1/6 |
| 2車単 | 二車勝単式 | 1-2着を着順通りに当てる | 1/72 | |
| 2車複 | 二車勝複式 | 1-2着を順不同で当てる | 1/36 | 2車単の1/2 |
| 2枠単 | 二枠勝単式 | 1-2着の枠番を着順通り | - | 9車立てのみ |
| 2枠複 | 二枠勝複式 | 1-2着の枠番を順不同 | - | 9車立てのみ |
| ワイド | 拡大二連勝複式 | 1-3着のうち2名を当てる | 3/36 | 的中範囲が広い |

### 初心者におすすめの車券

1. **ワイド**: 3着以内に入る2名を当てればいいので的中しやすい
2. **2車複**: 1-2着を順不同で当てる。ワイドの次にやさしい
3. **3連複**: 慣れてきたら挑戦。配当と的中率のバランスが良い

## オッズの見方

### オッズとは

オッズとは、車券が的中した場合に支払われる倍率。100円投票して5.0倍なら500円の払戻し。

### オッズから読み取れること

- **人気順**: オッズが低い（1倍台）＝多くの人がその組み合わせを買っている
- **本命**: オッズ1〜3倍台。勝つ可能性が高いと見られている
- **対抗**: オッズ4〜10倍台。本命に次ぐ評価
- **穴**: オッズ20倍以上。人気薄だが来る可能性がある
- **大穴**: オッズ100倍以上。来たら高配当

### オッズの活用

- **オッズの偏り**: 特定のラインに人気が集中 → そのラインが崩れれば高配当
- **意外な人気薄**: 実力があるのにオッズが高い選手 → 妙味あり
- **過剰人気**: 名前だけで買われている選手 → 過信は禁物

## 車券の買い方パターン

### フォーメーション

1着・2着・3着それぞれに候補を複数選び、その組み合わせを全通り買う方法。

```
例: 3連単フォーメーション
1着: 1, 3
2着: 1, 3, 5
3着: 1, 3, 5, 7
→ 全組み合わせを自動計算して購入
```

**メリット**: 効率的に網を張れる
**デメリット**: 点数が増えやすい

### ボックス

選んだ選手の全組み合わせを購入する方法。

```
例: 3連単ボックス 1-3-5-7
→ 4選手の3連単全順列 = 24通り
```

**メリット**: 取りこぼしが少ない
**デメリット**: 点数が多く、トリガミ（マイナス収支）になりやすい

### 流し

軸となる選手を決め、相手を複数選ぶ方法。

```
例: 3連単 1着流し
軸: 1番
相手: 3, 5, 7
→ 1-3-5, 1-3-7, 1-5-3, 1-5-7, 1-7-3, 1-7-5 = 6通り
```

**メリット**: 点数を抑えられる
**デメリット**: 軸が外れると全滅

## 資金配分の基本

### 均等買い

全ての買い目に同額を投票する方法。最もシンプル。

- **メリット**: 管理が簡単
- **デメリット**: 低配当の目に多く投票してしまう可能性

### 傾斜配分（メリハリ買い）

自信のある買い目に多く、押さえの目には少なく投票する方法。

- **メリット**: 的中時の回収を最大化
- **デメリット**: 判断を誤ると損失が大きい

### 資金管理のルール

1. **1レースの投資上限を決める**: 全資金の5〜10%が目安
2. **追い上げ（マーチンゲール）はしない**: 負けを取り返そうと倍額投資するのは危険
3. **トリガミを避ける**: 総投資額より払戻が少なくならないよう、買い目を絞る
4. **的中率と回収率のバランス**: 的中率50%でも回収率100%超えなら成功

## 予想から車券への変換

### ステップ1: 展開予想の結論を整理

展開予想から、有力な着順パターンを2〜3通り想定する。

### ステップ2: 軸となる選手を決定

最も信頼できる選手を軸にする。基準は以下。

- 有力ラインの先頭もしくは番手
- 競走得点が高い
- 直近の成績が安定している

### ステップ3: 買い目を組み立てる

展開パターンごとに買い目を整理する。

```
展開パターンA（本命）: 先行ラインが主導 → 1-3-5
展開パターンB（対抗）: 捲りが成功 → 7-1-3
展開パターンC（穴）  : もつれて混戦 → 5-9-1
```

### ステップ4: 点数と金額を調整

- 合計点数は10〜20点に抑える（3連単の場合）
- 本命パターンに厚く、穴パターンは薄く
- 払戻期待値が投資額を上回るか確認

## 車券のコツ

1. **全レース買わない**: 自信のあるレースだけに絞る
2. **情報を集める**: 出走表、選手コメント、バンク特性、天候
3. **感情で買わない**: 好きな選手だからではなく、根拠に基づいて判断
4. **記録をつける**: 予想と結果を記録し、自分の傾向を分析
5. **長期視点**: 1日の収支ではなく、月・年単位で収支を見る
