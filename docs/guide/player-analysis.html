脚質（きゃくしつ）

選手の戦い方のスタイルを表す、予想の最重要ファクターの一つです。

3つの脚質

脚質表記説明特徴
逃げ逃 先行選手。自らレースを引っ張る 残り1周半〜2周から先頭に立ち、押し切る。スタミナと脚力が必要
追い込み追 番手・三番手で戦う選手 ライン先頭の後ろにつき、最終直線で差し切る。判断力と瞬発力が重要
自在両 先行も追い込みもできる選手 展開に応じて戦い方を変えられる。器用さが武器

脚質の見極めポイント

  • B（バック回数）が多い → 先行力がある
  • 競走得点が高く、Bが少ない → 追い込みで結果を出している
  • 決まり手に「逃げ」「捲り」が多い → 自力型
  • 決まり手に「差し」「マーク」が多い → 追い込み型
脚質と車券の関係
「逃」の選手は1着か着外が多く、ハイリスク・ハイリターン。「追」の選手は2・3着に安定しやすく、2車複やワイドと相性が良い傾向があります。

競走得点の見方

競走得点は選手の実力を測る最も客観的な指標です。直近4ヶ月の成績から算出されます。

得点の目安

得点帯級班目安レベル
120点以上SS級トップ中のトップ
110〜119点S1上位一線級
100〜109点S1〜S2S級標準
90〜99点S2〜A1中堅
80〜89点A1〜A2A級上位
70〜79点A2〜A3A級標準

得点の活用ポイント

  • 同一ライン内の比較 - ライン先頭の得点が高いほど、そのラインは有力
  • 級班をまたぐ比較には注意 - S級とA級では対戦相手のレベルが違うため、単純比較は不可
  • 得点の推移 - 上昇中の選手は好調、下降中は不調のサイン

決まり手の理解

競輪の決まり手（勝ち方）を理解することは、レース展開の予想に直結します。

主な決まり手

決まり手説明選手タイプ
逃げ先頭で逃げ切って1着先行選手
捲り後方から一気にまくって1着自力選手
差し最終直線で前の選手を差して1着追い込み選手
マークライン先頭について行き1着追い込み選手

決まり手と車券

  • 逃げが決まりやすいバンク → 先行選手のラインを中心に買う
  • 差しが決まりやすいバンク → 番手選手（ライン2番手）に注目
  • 捲りが決まりやすい展開 → 後方待機の自力選手に注意

選手の調子・状態の見方

直近成績のチェックポイント

  1. 直近4走の着順 - 安定して上位に来ているか
  2. 決まり手の傾向 - いつもの戦い方ができているか
  3. 競走得点の推移 - 上昇・下降・横ばい

コンディション要素

要素チェック内容
落車・怪我直近に落車している選手は状態が万全でない可能性
連闘（れんとう）短期間に連続出走している場合、疲労に注意
ホームバンク自場では走り慣れているため有利な傾向

選手コメントの読み方

コメント例読み取り
「脚の状態がいい」「感触がいい」好調アピール
「前で勝負」「先行する」先行宣言（ラインの並びに影響）
「流れで」「臨機応変に」展開次第で動く意思表示
「連れ込む」「番手から」追い込みに徹する宣言
