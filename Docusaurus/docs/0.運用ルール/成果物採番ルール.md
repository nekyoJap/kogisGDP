
以下命名規則によって成果物の採番を行う。

```auto
<成果物3桁><ドメイン2桁><BC2桁><連番3桁>_<成果物名>(_<ドメイン名>_<BC名>※(ドメイン/BC単位で作成する成果物の場合は付与)
```
| 種類 | 説明 | 例 |
|----|----|---|
| カテゴリ | ビジネス要件検討やBC単位検討など成果物作成フェーズ | BR（ビジネス要件検討/Business Requirement） |
| ドメイン | 審査発行管理、顧客管理などドメイン定義 | CM（顧客管理/Customer Management） |
| BC | ドメイン内で境界定義されたコンテキスト(BC)定義 | CV（顧客審査/Customer Verification |
